The following tables list the members exposed by the TwoHandsInteractable type.

Methods
 
Name Description
Awake()
OnSelectEntered(SelectEnterEventArgs) When the first hand selects the object, the pivot is positioned on the hand, then the pivot is settled as parent of the object, to make it follow. When the second hand selects the object, the pivot is rotated using the angle from hand1 to hand2.
OnSelectExited(SelectExitEventArgs) When both the hands leave the object, the hierarchy is restored, making the pivot child of the object. When only one hand leave the object, the pivot is settled to the remaining hand.
